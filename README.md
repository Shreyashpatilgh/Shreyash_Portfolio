Personal Portfolio Website:

This is my personal portfolio web application built using Streamlit.
It showcases my profile, skills, projects, education and certifications in a clean and interactive way.

 Features:

Modern single-page portfolio layout

Responsive navigation (Home, Education, Skills, Projects, Certifications)

Downloadable resume

Project section with GitHub links

Clickable certification links

Clean UI with custom CSS styling

Built entirely using Python + Streamlit

 Tech Stack:

Python

Streamlit

HTML & CSS (custom styling)

Git & GitHub
